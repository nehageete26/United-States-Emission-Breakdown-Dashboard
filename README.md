# 🌍 Emissions Dashboard (Databricks)

An interactive data analytics dashboard built using Databricks to visualize and analyze carbon emissions across the United States.

## 📊 Overview

This project provides a comprehensive breakdown of emissions across the U.S. using visual analytics. It helps identify high-emission regions, population impact, and state-wise contribution to overall emissions.

## 🚀 Features

- 📍 Geospatial visualization of emissions across the continental U.S.
- 📈 Emissions vs Population scatter analysis
- 🥧 Top 10 states contribution (percentage breakdown)
- 📊 County-level emission comparison (bar chart)
- ⚡ Interactive dashboard built on Databricks SQL

## 🛠️ Tech Stack

- Databricks (SQL Dashboard)
- SQL
- Data Visualization
- Geospatial Analytics

## 📁 Dataset

- Source: EPA (Environmental Protection Agency) - 2023 dataset
- Contains:
  - State & County-level emissions
  - Population data
  - Emissions per capita

## 📌 Key Insights

- A small number of states contribute to a large percentage of total emissions.
- Higher population regions tend to have lower emissions per capita.
- Certain counties dominate total emission output.

## ⚙️ How to Run

1. Upload dataset to Databricks workspace
2. Create tables using SQL
3. Build queries for:
   - Emissions aggregation
   - Population comparison
4. Use Databricks Dashboard to create visualizations
5. Combine all widgets into a dashboard
